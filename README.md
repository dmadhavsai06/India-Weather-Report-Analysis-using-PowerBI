# 🌦️ India Weather Dashboard (Power BI)

An interactive Power BI dashboard designed to analyze weather patterns across Indian cities. The dashboard provides insights into temperature, rainfall, atmospheric pressure, and geographic weather distribution through interactive visualizations.

---

## 📊 Dashboard Overview

![Dashboard Overview](screenshots/Dashboard%20Overview.png)

---

## 🚀 Features

- Interactive City-wise Filtering
- Geographic Weather Distribution Map
- Average Temperature Monitoring
- Maximum Temperature Analysis
- Monthly Rainfall Trend Analysis
- Atmospheric Pressure Analysis
- Maximum vs Minimum Temperature Comparison
- Dynamic Dashboard Interactions

---

## 🛠️ Tools & Technologies

- Power BI
- DAX (Data Analysis Expressions)
- Data Visualization
- Data Modeling
- Data Analysis
